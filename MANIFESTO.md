# The Arcane Manifesto

> *Tercera llamada… comenzamos.* *(Third call… we begin.)*

1. **Your project's memory belongs to you.** Decisions, plans, and session
   journals live in your repo as plain files you own — not in an AI vendor's
   cloud. Switch tools whenever you want; the memory travels with you.

2. **Work with the AI you choose.** Define your agents once; Arcane puts them
   exactly where Claude, Copilot, and Codex each expect to find them. The
   framework is loyal to you, not to a model vendor.

3. **Trust is earned, level by level.** Every agent starts with the least
   power that does its job, and is promoted only after it proves itself. Six
   named levels, set per agent, per repository.

4. **Standards install like packages.** Your team's rules — git conventions,
   testing bars, agent policy — are versioned files you install, update, and
   remove like any other dependency.

5. **Playful names, plain instructions.** The characters and commands carry
   the magic; everything they actually do is written in clear, professional
   English.

6. **The more a command can do on its own, the clearer its name must be.**
   An autonomous command's name is a contract: it says exactly what it will
   do.

7. **The defaults are safe; the universe is yours.** Arcane's built-in
   personas can ship anywhere, and the whole roster re-skins into any
   universe you love — with the publishing checklist already in the box for
   the day you go public.

8. **Unwritten memory is just aspiration.** Decisions, plans, and checks
   live in files; nothing ships on recollection — human or agent.

9. **Rules that aren't enforced are just suggestions.** What the documents
   declare, the platform should gate.

10. **Your files are never overwritten.** Arcane skips what you wrote,
    previews before it deletes, and proposes changes to other tools'
    settings instead of editing them.

11. **Defined once, shown everywhere.** An agent's identity lives in one
    file; every screen, document, and client reads from it, and none of
    them keeps its own copy.

12. **It must pass the brother test.** If someone non-technical is afraid
    to press the button, the button is wrong — not the person.

---

**The game is the tool.** *The generation that grew up inside worlds will
run their work through one.*
